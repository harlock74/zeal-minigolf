# zeal-minigolf

zeal-minigolf is a mini-golf type game for the Zeal 8-bit Computer.

It runs at **640×480 @8bpp**

## Rules


## Controls


- `LEFT` `RIGHT` (or SNES D-pad): to aim
- `SPACE` (or SNES `B`): increase / decrease power to shoot
- `RIGHT_SHIFT` (or SNES `SELECT`): quit
